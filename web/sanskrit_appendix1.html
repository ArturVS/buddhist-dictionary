Certain rules that modify the writing of combinations of consonants in Sanskrit. For example, the word pañca (English: five) combines the letters प (pa), ञ (ña), and च (ca). The a in ञ (ña) is not pronounced. To express this the vertical line in ञ (ña) is omitted so that pañca is written in Devanagari as पञ्च. Other examples of consonant combinations that can be combined by removing a vertical bar are:

  • ggha: ग (ga) + घ (gha) is written as ग्घ
  • ghya: घ (gha) + य (ya) is written as घ्य
  • tsa: त (ta) + स (sa) is written as त्स
  • tsya: त (ta) + स (sa) + य (ya) is written as त्स्य
  • nma: न (na) + म (ma) is written as न्म

A second kind of example occurs with the word aṅgaṃ (English: limb), which combines the letters अ (a), ङ (ṅa), and ग (ga) ending with the Anusvāra (nasalization) diacritic ṃ. The letter ङ (ṅa) does not have a vertical bar to remove so it is combined with the letter ग (ga). The leads to writing the word aṅgaṃ as अङ्गं. Other cases where this same method is applied are when the following consonant does not provide a convenient point to anchor from or the vertical bar cannot easily be detached. Examples are shown below.

  • ṅga: ङ (ṅa) + ग (ga) is written as ङ्ग
  • ṭṭha: ट (ṭa) + ठ (ṭha) is written as ट्ठ
  • hla: ह (ha) + ल (la) is written as ह्ल
  • ñca: ञ (ña) + च (ca) is written as ञ्च
  • ṣṭa: ष (ṣa) + ट (ṭa) is written as ष्ट

There are some other exceptions for combinations of त (ta) and क (ka). In these situations a small loop is added resulting in त् (tta) or क्त (kta). These are shown below.

  • tta: त (ta) + त (ta) is written as त्त
  • kta: क (ka) + त (ta) is written as क्त

An open form is used for the letter (ma), as shown below.

  • dma: द (da) + म (ma) is written as द्म

Combinations with र (ra) as the first consonant are written with a curly diacritic above the second consonant. This is shown in the list below.

  • rta: र (ra) + त (ta) is written as र्त
  • rya: र (ra) + य (ya) is written as र्य
  • rka: र (ra) + क (ka) is written as र्क

In combinations where र (ra) as the second consonant a short horizontal line is used, as shown below.

  • pra: प (pa) + र (ra) is written as प्र (pra)
  • tra: त (ta) + र (ra) is written as त्र (tra)
  • kra: क (ka) + र (ra) is written as क्र

The consonant combination kṣa is a special case. It is written as क्ष्. For example, the name of the goddess Lakṣmī ल (la) + क (ka) + ष (ṣa) + मी (mī) is written as लक्ष्मी. The consonant combination jña is another special case. It is written as ज (ja) + ञ (ña) = ज्ञ (jña).

When consonants are doubled then similar rules for writing the combinations are applied. For examlpe, gg, componsed of ग (ga) + ग (ga) is written as ग्ग. However, if an aspirated consonant is repeated then the first in the pair is changed to an unaspirated form. For, example, घ (gha) + घ (gha) is written as ग्घ (ggha). This is really a modification of the phonetic rule rather than the rule for writing the script.

The results may look different with different fonts. There is a full list of consontant combinations at the Sanskrit Documents, Conjungate Consonants page (see References).
