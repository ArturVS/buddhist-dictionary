Frequently Asked Questions

The Fo Guang Shan Nan Tian Institute Chinese-Sanskrit-English Buddhist Dictionary and language toolkit (佛光山南天大學中梵英詞典與語言分析工具) is a non-profit, open source project for use in reading and analyzing Chinese and Sanskrit Buddhist texts.

Goals

The goals the dictionary are

  1. Create an online dictionary that is easy to use for everybody interested in Buddhism, including lay people reading Buddhist texts, students, translators, and academics. If you are looking at a large chunk of Chinese or Sanskrit text but not sure what it means then this is for you.
  2. Create electronic tools that are useful for lingustic analysis of Buddhist texts, including identification of specialist Buddhist terms and comparison of Chinese and Sanskrit texts. If your goal is to translate the Mahayana Buddhist Canon, then this website and GitHub project is especially for you.
  3. Share the source code and content with the general public. If you want to reuse source code or contribute source code or data then the GitHub project is built for you.

What the dictionary is not includes: an encyclopedia or a glossary of Buddhist terms. Sorry, but there are people better qualified than me to create this. Please see the Resources section for some excellent resources. The project started out as a generic Chinese dictionary and is OK for that. Most of the terms in the dictionary are actually not Buddhist terms. I hope that people that used the dictionary before I specialized it to Buddhism keep using it.
