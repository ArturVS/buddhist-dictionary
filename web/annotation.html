Text Management, Annotation, and Gloss

This page gives some background on the data and bdictutil.py command line tool that was used to generate the data that drives this web site.

Background

The collection of texts in this web site are managed as a text corpus. A text corpus, or simply corpus, is a term used in linguistics to mean a representative collection of texts used to study the linguistic characteristics of a language. The Brown Corpus, first published in 1961 [Francis 1964], was the first large corpus developed. It consists of about 1 million English words.

An annotated corpus or treebank adds information about the structure of the text, particularly part of speech and semantic information. The Penn Treebank Project [Penn Treebank Project] was the first large-scale treebank developed. The NTI Chinese-Sanskrit-English Buddhist Dictionary is just beginning to include parsed text in the form of a phrase dataset. The use of a phrase dataset consisting of short phrases can be used in a very practical way in the Buddhist texts because:

  1. Most Buddhist literature is written in Literary Chinese, starting in the first century CE to the present day and remains untranslated.
  2. Phrases in the text are repeated frequently. For example, the phrase 於意云何 <what do you think?> is repeated 30 times in the Diamond Sutra, which contains a total of only 4615 words.
  3. The Chinese language has a long history during which words have acquired many meanings resulting in a great deal of ambiguity when looking up words in the dictionary. The context of the word is needed to determine word sense and the simplest way to to determine context is to check if the word is used in a similar phrase with a known translation.
  4. Short phrases can be easily matched into untranslated sentences.

Each corpus entry includes meta data about a primary source document, such as the source and links to plain text versions, English translations, vocabulary analysis, and other information.

Gloss is a form of annotation that explains the meanings of words in texts. A collection of glosses is called a glossary. A historic example is Biblical gloss that was used to annotate the meanings of words in the Bible. Biblical gloss was usually added in margins and included commentaries as well. Interlinear gloss is gloss added between the lines. Historic annotations are valuable in themselves and may give explainations of the source text that would not be known otherwise. However, they have often been ommitted when documents were new versions of documents were published, retaining the original text only.

Gloss is also used in a modern context by linguists to help them study and discuss languages that they cannot understand themselves. Typically, this kind of gloss in added after the native terms in square brackets, such as 佛 [Buddha]. Besides the meaning of foreign words in English, linguists also sometimes indicate pronunciation with gloss. The Penn Chinese Treebank has a particular style of gloss used that includes gloss for indivual words in square brackets and angle brackets for the phrase as a whole [Nianwen 2005]. For example, 乘 [riding] 白 [white] 象 [elephant] <riding a white elephant>. This web site follows that style in the phrase memory dataset, putting in mouse-over for the convenience of readers. You can see the Penn Treebank format gloss used in the phrase detail pages on this web site. For example, 乘白象.

Use of markdown and corpus management

Each corpus entry has a text file that uses a markdown file. Markdown is a plain Markdown can allow people who are not software or web design specialists to easily work with plain text that can be transformed to HTML later. Markdown consists of simple and intuitive patterns, such has headers and bullet lists for making plan text more readable. This makes it easy to add annotations to primary source documents or copies of them. See the Buddhist Dictionary GitHub project corpus directory for a list of markdown files used in the project.

The collection of primary texts and markdown documents is through a corpus schema, including start and end markers for primary text. Adaptations for easy display of primary source text from common repositories of primary sources, particulary CBETA. For details on the corpus schema see the corpus-readme.txt document.

Markdown viewers and parsers widely available, so it is a good medium for transfering text between systems. In particular, users of the web site can use the markdown generated on the web site, copying that elsewere to to another site. The terms and conditions of the the web site allow direct copying, with attribution.

Part-of-Speech Tagging

This is a new effort. At the moment, part-of-speech (PoS) tagging is only used in the phrase dataset. Initially, only commonly occuring phrases identified through ngram scanning will be tagged.

It follows the PoS tagging format documented by the Penn Chinese Treebank project. That was designed for modern Chinese but this project is mainly in literary Chinese, which has a different grammar. Still, the Penn format is used as the best choice at the moment. See the Penn Chinese Treebank web site for more details. See the Part-of-Speech Tag Definitions for a list of the tags used in this project.
