About the Project

The Fo Guang Shan Nan Tian Institute (NTI) Chinese-English Dictionary and Buddhist Text Project (佛光山南天大學中英詞典與佛教文本項目) is a non-profit, open source project for use in reading and analyzing Chinese and Sanskrit Buddhist texts.

Goals

The goals the dictionary are

  1. Create an online dictionary that is easy to use for everybody interested in Buddhism, including lay people reading Buddhist texts, students, translators, and academics. If you are looking at a large chunk of Chinese or Sanskrit text but not sure what it means then this is for you. See notes on the Casual use for a private translation workflow on ways that you can use the web site for this purpose.
  2. Create electronic tools that are useful for lingustic analysis of Buddhist texts, including identification of specialist Buddhist terms and comparison of Chinese and Sanskrit texts. If your goal is to translate the Mahāyāna Tripiṭaka, then this website and GitHub project is especially for you. See the workflow described in the section Project team member for a publicly available translation.
  3. Share the source code and content with the general public. If you want to reuse source code or contribute source code or data then the GitHub project is built for you.
